HTTP Request from Node.JS

Today we'll look at the Request library, a simple HTTP(S) client for Javascript, and explore JSON objects.

Setup

You will need to prepare your working directory.

cd ~
mkdir tut2
cd tut2
npm init -y
npm install request --save
touch mycurl.js

Fireball API

Let us look at the Fireball Data API. It gives us information about fireballs, or meteors, as observed by NASA. It's a very simple API, with three GET methods.

Code

Use the snippet of code below to get you started. Use it to edit the file mycurl.js This provides the basic skeleton for making your requests to any external HTTP server.

const request = require('request')

request.get('URL-HERE', function (err,res, body) {
if(err) {
console.log(err)
} else {
data = JSON.parse(body)

//Modify this to access different JSON fields
console.log(data)
}
})

Try running the code using Node.js:

node mycurl.js
